# ReproducibleComparativeStudyofMachineLearningClassifiersHeartDiseasePrediction
Cardiovascular disease (CVD) causes ~18M deaths annually. This study compares 8 machine learning models on a dataset of 1,025 patients. Random Forest achieved the best performance (accuracy 0.997). Key features like ca, thalach, oldpeak, thal, and cp were most significant for prediction.
